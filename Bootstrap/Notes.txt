What is Bootstrap?

Bootstrap is a free, open source, framework developed by twitter - the most popular HTML, CSS, and JS framework for developing responsive, mobile first projects on the web (websites that are coded to be fully accessible on mobile devices)

Bootstrap comes in to play when you're trying to build templates

Cuts down on development time by avoiding having to re-code things like buttons, tables, tabs, forms, tables, navigation, alerts, etc.

Bootstrap Grid System https://www.youtube.com/watch?v=GDwWmrpCa30

Bootstrap Grid system is used for creating page layouts through a series of rows and columns. The Grid System consists of 12 columns.

Bootstrap Grid Classes

Devince                                            Bootstrap Class
Extra small devices - mobile phones (<768px)       .col-xs-*
Small devices - tablets (>= 768px)                 .col-sm-*
Medium devices - Desktops (>=992px)                .col-md-*
Large devices - Desktops (>=1200px)                .col-lg-*
